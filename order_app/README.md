## Features

* Ordering drinks, foods by table.
* Checkout, preview & print invoice.
* Profile feature.
* Manage drink, food & category.
* Manage table.
* Manage account.
* Sales report.

## Install

Clone this project:
```
$ git clone https://github.com/jade28/starbucks-coffee.git
```
cd `starbucks-coffee`:
```
$ cd starbucks-coffee
```
Install packages for `flutter` apps:
```
$ flutter packages get
```
